任务1里说的「**按类别筛选**」，对应页面上的 **左侧侧边栏 `Categories` 这一列的分类按钮**。

- **页面元素位置**：主界面左侧（宽屏时是左栏），标题是 **`Categories`**，下面一行行的分类（例如 `All`、以及各个 tag 名称）。

- **代码对应关系**：
  - 分类 UI 在 `src/components/Sidebar.vue`，每个分类是一个 `<button @click="emit('select', category)">`（点击即筛选）。
  - `src/App.vue` 里用 `<Sidebar ... @select="handleCategorySelect" />` 接收点击，并更新 `activeCategory`，再通过 `filteredResources` 做过滤。

如果你现在页面上没看到左侧栏，通常是窗口太窄导致布局变化；把窗口拉宽一点（进入两列布局）就能看到 `Categories` 那块了。
